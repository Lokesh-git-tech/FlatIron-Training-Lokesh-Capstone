Knowledge Base ID: KB-1003
Category: Identity
Severity: Low

Title
Password Reset Issues

Issue Summary
Password reset not reflected.

Symptoms
- Login failures

Possible Causes
- Replication delay
- Cached credentials

Resolution Steps
1. Wait 15 minutes
2. Sign out
3. Clear cache
4. Retry

Verification
Login succeeds.

Escalation Criteria
Escalate to Identity Team.
