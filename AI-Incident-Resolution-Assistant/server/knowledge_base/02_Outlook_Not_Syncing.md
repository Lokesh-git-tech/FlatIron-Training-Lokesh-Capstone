Knowledge Base ID: KB-1002
Category: Email
Severity: Medium

Title
Outlook Not Syncing

Issue Summary
Outlook is not synchronizing new emails.

Symptoms
- Inbox not updating
- Send/Receive errors

Possible Causes
- Offline mode
- Network issue
- Mailbox quota

Resolution Steps
1. Verify internet
2. Disable Offline Mode
3. Restart Outlook
4. Repair profile

Verification
New emails received.

Escalation Criteria
Escalate to Messaging Team.
