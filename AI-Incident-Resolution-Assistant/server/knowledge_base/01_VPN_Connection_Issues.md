Knowledge Base ID: KB-1001
Category: Network
Severity: Medium

Title
VPN Connection Issues

Issue Summary
Users cannot connect to the corporate VPN.

Symptoms
- VPN authentication failed
- Connection timeout
- Disconnected immediately

Possible Causes
- Password recently changed
- Cached credentials
- VPN service unavailable

Resolution Steps
1. Disconnect VPN
2. Clear saved VPN credentials
3. Restart VPN client
4. Sign in using latest password
5. Verify MFA

Verification
User connects successfully.

Escalation Criteria
Escalate to Network Operations after 15 minutes.
